## ✨ Features

- 📱 **Responsive Design** — Works on all devices (desktop, tablet, mobile).  
- 🌙 **Dark Mode Toggle** — Switch between light and dark themes.  
- 🖥️ **Project Showcase** — Display your projects with links and descriptions.
